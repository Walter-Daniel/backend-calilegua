# Proyecto de Bootcamp NestJS - Calilegua

## Descripción

Este proyecto es parte del Bootcamp de NestJS en Calilegua. Tiene como objetivo demostrar los conceptos y prácticas fundamentales aprendidos durante el curso, incluyendo controladores, servicios, pipes, entidades y Objetos de Transferencia de Datos (DTOs).

## Tecnologías Utilizadas

- **NestJS**: Un framework progresivo de Node.js para construir aplicaciones del lado del servidor de manera eficiente y escalable.
- **TypeScript**: Un lenguaje de programación con tipado fuerte que se basa en JavaScript.

## Características

- **Controladores**: Implementación de endpoints API para manejar solicitudes entrantes y devolver respuestas.
- **Servicios**: Implementación de la lógica de negocio para gestionar datos e interactuar con entidades.
- **Pipes**: Transformación y validación de datos para las solicitudes entrantes.
- **Entidades**: Definición de la estructura del modelo de datos utilizado en la aplicación.
- **DTOs**: Objetos de Transferencia de Datos para la validación de datos y verificación de tipos.
